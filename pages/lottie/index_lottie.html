About Lottie:

Lottie is an open-source animation framework that allows you to render high-quality 2D animations similar to Rive.

Properties:

  • Free to use and open-source
  • Provides a plugin that can work in Figma, Framer and other softwares
  • Uses JSON or .lottie format for lightweight animations
  • Cross-platform support (Web, iOS, Android, React, Flutter, etc.)
  • Scalable vector-based animations with no quality loss
  • Supports looping, autoplay, and playback control via JavaScript
  • Compatible with After Effects through the Bodymovin plugin
